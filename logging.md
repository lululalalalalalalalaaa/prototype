# 작업 로그 (logging.md)

이 문서는 "**사용자가 어떤 프롬프트로 무엇을 요청했고, 그에 따라 어떤 작업·결정·검증을 했는지**"를
시간순으로 기록합니다. 수치는 모두 실측값입니다.

> 핵심 원칙(전 과정 공통): **measure-driven** — 변경마다 골든셋 eval로 before/after를 수치 비교하고,
> 인프라성 변경은 "지표 불변"을 성공 기준으로 삼는 **동작 보존**을 검증했습니다.

## 누적 결과 요약

- **검색 품질** (확장 골든셋 **105문항**: 90 정답 + 15 no_match, k=5, HWP 정제+구조청킹+이미지 후):
  - Dense 0.811 → Hybrid 0.794 → **Rerank MRR 0.972**(2회 동일). 난이도별: easy 0.980 / medium 0.978 / **hard 0.950**.
  - **발견**: BM25(하이브리드) 이득 미미(리랭커가 흡수) / 정제·구조청킹은 dense 살짝 낮춰도 rerank는 0.962→0.972로 올림.
- **그라운딩**(전체 파이프라인): off-domain 기권 **1.000**(15/15), 응답 정확도 **0.962**(경계 1건), 과잉기권 0.026.
- **내용 질의**(이름 아닌 공정·연료·기능단위 12문항): 추천 정확도 **12/12**.
- **인제스션**: HWP 폼 노이즈 정제 + 섹션·표 분리 청킹(본문452+표431) + 공정흐름도 이미지 vision 설명(29).
- **관측성·출처·비용**: 단계별 trace(타이밍·토큰·USD), 추천 근거를 **섹션명+위치**로 인용, 서버 콘솔 + 인앱 패널.
- **인덱스**: 37 문서 / 912 청크 / 메타데이터 37. **public repo에 커밋·푸시**.
- **테스트**: 139 passed (전부 API 키 불필요, mock). cspell 0 이슈.
- **부수 정리**: NFD 파일명 버그 수정, `.env` 우선순위, cspell 사전, main.py/lci_reports.json/ingest.pipeline 제거.

## 반복적으로 작동한 패턴

- **측정 후 결정**: BM25 v1 회귀(0.871)·Stage 6 프록시 오측정(0.375)·**호출 병합 회귀(MRR 0.898·기권 0.800)**를 eval이 잡아내, 잘못된 변경을 막음(부정 결과도 자산).
- **동작 보존 마이그레이션**: 리팩터·아티팩트 전환을 "eval 지표 불변"으로 안전 검증.
- **YAGNI/over-engineering 차단**: 표 파싱·ANN·프롬프트 강화를 *실측 근거로* 제외.
